Définition du HTML et du CSS

HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. HTML describes the structure of a web page semantically.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used for describing the look and formatting of a document written in HTML. CSS is used to control the layout of multiple web pages all at once, customizing aspects such as layout, colors, and fonts.
